## Top Picks

**Rohit Sinha, Evgenios Kornaropoulos, Thang Hoang, Sajin Sasy, Sam Kumar, Nuno Santos** — Sinha published on oblivious computation inside enclaves. Kornaropoulos and Hoang are leaders in oblivious primitives and encrypted databases. Sasy co-built TEE-based oblivious systems (Opaque). Kumar built MAGE exploiting oblivious memory patterns. Santos published on TEE-based secure query processing.
